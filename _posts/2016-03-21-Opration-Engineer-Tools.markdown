### [How to see time stamps in bash history](http://askubuntu.com/questions/391082/how-to-see-time-stamps-in-bash-history)

1.  ```
    echo 'export HISTTIMEFORMAT="%d/%m/%y %T "' >> ~/.bash_profile ; source ~/.bash_profile
    ```

2.  ```
    history
    ```
